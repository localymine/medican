This plugin provided users to easily add a custom user registration & login  form anywhere on their site using simple shortcode.

== Description ==
This plugin creates a user interface for adding custom registration forms to any post or page on your WordPress website. Simply navigate to the edit page for post .
Admin Custom Login & Registration Form plugin give ability to customize your WordPress login & registration Form page according to you. 

> **provided Shortcode**
>
> * login form Shortcode: [cflrf_custom_login_form] 
> * Registration Form Shortcode: [cflrf_registration_form]
>

**Live Demo:** [Custom Login & Registration Form Live Demo](http:http://dropsofvisions.com/)


> **Standard Features**
>
> * Add Your Custom Login Form any post/pages.
> * Login Form Colour
> * Login Form Font Colour
> * Login Form Font Size
> * Login Form Button Colour
> * Login Form Button Size
> * Login Form with Font Awesome Icons
>

== Installation ==

1. Download ` Custom  Frontend  Login Registration Form` and upload contents to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Insert forms using the Shordcode registration button on your post/page.



### Responsive Design:
Plugin Design is totally mobile compatible.

Plugin is integrated with 500+ google fonts. So add beautiful font styles fro you form panel.
